# Domain 配置文件模板

创建新领域配置时，复制此模板并填充内容。

## 最小可用标准（填写前检查）

- [ ] `trigger-keywords` 至少 5 个（确保路由准确命中）
- [ ] `schema-version` 为 1（当前版本）
- [ ] 知识域至少 3 个（覆盖该领域核心方向）
- [ ] 每个知识域高频题至少 3 道（标注频率）
- [ ] 记忆锚点示例至少 3 个（展示领域特色锚点风格）
- [ ] 技术基线覆盖至少 3 个 level

## 填写说明

| Section | 目的 | 字数/数量建议 |
|---------|------|--------------|
| trigger-keywords | 自动路由命中 | 5-15 个，含中英文变体 |
| 角色描述 | 设定面试官画像 | 2-3 句话 |
| 技术基线 | 按 level 区分难度 | 覆盖实习→5年+ |
| 领域专用约束 | 避免生成错误 | 3-5 条硬规则 |
| 知识域 | 题目分类依据 | 3-10 个域 |
| 高频题参考 | 大纲阶段参考 | 每域 3-8 道 |
| 记忆锚点示例 | 展示锚点风格 | 3-5 个 |

---

```yaml
trigger-keywords: [关键词1, 关键词2, 关键词3, 关键词4, 关键词5]
schema-version: 1
```
