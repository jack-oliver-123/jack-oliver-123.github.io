## 题目入口

题目链接：[矩阵置零](https://leetcode.cn/problems/set-matrix-zeroes/)

## 一句话读懂题目

给你一个 `m × n` 的矩阵，如果某个元素是 `0`，就把它所在的整行和整列都置为 `0`。

要求原地操作。

比如：

```text
matrix = [[1, 1, 1],
          [1, 0, 1],
          [1, 1, 1]]
```

`(1, 1)` 位置是 `0`，把第 1 行和第 1 列全部置零：

```text
[[1, 0, 1],
 [0, 0, 0],
 [1, 0, 1]]
```

## 先用生活例子理解

想象你在教室里找到几个"传染源"（值为 0 的格子），每个传染源会让自己所在的整行和整列都"感染"。

但你不能一找到就立刻置零——因为新产生的零会引发连锁反应，把不该清零的位置也清掉。

正确做法是：先全部"标记"哪些行和列要清零，最后统一清零。

## 为什么不能边找边置零

看这个例子：

```text
[[1, 0],
 [1, 1]]
```

`(0, 1)` 是 0，应该把第 0 行和第 1 列置零。结果应该是：

```text
[[0, 0],
 [1, 0]]
```

如果你一发现 `(0, 1)` 是 0 就把第 0 行全置零，那 `(0, 0)` 就变成 0 了。接下来扫描到 `(0, 0)` 是 0（但这是你刚设的），就会把第 0 列也清零——结果全变成 0 了。

所以必须先标记再操作。

## 方法一：额外数组标记（好理解）

用两个数组分别记录哪些行、哪些列需要置零：

```text
第一遍：扫描矩阵，找到所有 0，标记对应的行和列
第二遍：扫描矩阵，如果当前行或列被标记了，就置零
```

```java
class Solution {
    public void setZeroes(int[][] matrix) {
        int m = matrix.length;
        int n = matrix[0].length;
        boolean[] rowZero = new boolean[m];
        boolean[] colZero = new boolean[n];

        // 第一遍：标记
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (matrix[i][j] == 0) {
                    rowZero[i] = true;
                    colZero[j] = true;
                }
            }
        }

        // 第二遍：置零
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (rowZero[i] || colZero[j]) {
                    matrix[i][j] = 0;
                }
            }
        }
    }
}
```

时间 `O(m*n)`，空间 `O(m+n)`。

## 方法二：用第一行第一列做标记（O(1) 空间）

核心思路：不额外开数组，直接用矩阵的第一行和第一列来当标记位。

但第一行和第一列本身也可能需要置零，所以需要两个额外变量记录它们是否要清零。

### 步骤

1. 先检查第一行和第一列本身有没有 0，用两个布尔变量记录；
2. 用第一行和第一列作为标记位：遍历剩余部分，如果 `matrix[i][j] == 0`，就把 `matrix[i][0]` 和 `matrix[0][j]` 设为 0；
3. 根据标记位把剩余部分置零；
4. 最后根据步骤 1 的记录决定是否把第一行/第一列置零。

## 手动走一遍例子

```text
matrix = [[0, 1, 2, 0],
          [3, 4, 5, 2],
          [1, 3, 1, 5]]
```

**步骤 1**：检查第一行有 0 → `firstRowZero = true`；检查第一列有 0 → `firstColZero = true`。

**步骤 2**：遍历 `i=1..2, j=1..3`，没有找到 0。标记位不变。

**步骤 3**：根据标记位清零（第一行 `matrix[0][1]=1, matrix[0][2]=2` 不是 0，第一列 `matrix[1][0]=3, matrix[2][0]=1` 不是 0），剩余部分不需要置零。

**步骤 4**：`firstRowZero = true` → 第一行全部置零；`firstColZero = true` → 第一列全部置零。

结果：

```text
[[0, 0, 0, 0],
 [0, 4, 5, 2],
 [0, 3, 1, 5]]
```

## Java 代码（O(1) 空间）

先记住一句话：

> 借用第一行第一列当"便签纸"，先标记后清零，最后处理便签纸本身。

```java
class Solution {
    public void setZeroes(int[][] matrix) {
        int m = matrix.length;
        int n = matrix[0].length;
        boolean firstRowZero = false;
        boolean firstColZero = false;

        // 步骤 1：检查第一行和第一列是否有 0
        for (int j = 0; j < n; j++) {
            if (matrix[0][j] == 0) {
                firstRowZero = true;
                break;
            }
        }
        for (int i = 0; i < m; i++) {
            if (matrix[i][0] == 0) {
                firstColZero = true;
                break;
            }
        }

        // 步骤 2：用第一行第一列做标记
        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                if (matrix[i][j] == 0) {
                    matrix[i][0] = 0;
                    matrix[0][j] = 0;
                }
            }
        }

        // 步骤 3：根据标记置零（从 1 开始，不动第一行第一列）
        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                if (matrix[i][0] == 0 || matrix[0][j] == 0) {
                    matrix[i][j] = 0;
                }
            }
        }

        // 步骤 4：处理第一行和第一列
        if (firstRowZero) {
            for (int j = 0; j < n; j++) {
                matrix[0][j] = 0;
            }
        }
        if (firstColZero) {
            for (int i = 0; i < m; i++) {
                matrix[i][0] = 0;
            }
        }
    }
}
```

## 代码逐行拆解

```java
boolean firstRowZero = false;
boolean firstColZero = false;
```

记录第一行/第一列自身是否需要置零。这两个变量是整个方法能用 `O(1)` 空间的关键。

```java
for (int j = 0; j < n; j++) {
    if (matrix[0][j] == 0) {
        firstRowZero = true;
        break;
    }
}
```

扫描第一行，只要有一个 0，整行最终都要清零。

```java
for (int i = 1; i < m; i++) {
    for (int j = 1; j < n; j++) {
        if (matrix[i][j] == 0) {
            matrix[i][0] = 0;
            matrix[0][j] = 0;
        }
    }
}
```

扫描剩余部分（从第 1 行第 1 列开始）。发现 0 时，把标记写到第一列和第一行。

```java
for (int i = 1; i < m; i++) {
    for (int j = 1; j < n; j++) {
        if (matrix[i][0] == 0 || matrix[0][j] == 0) {
            matrix[i][j] = 0;
        }
    }
}
```

根据标记清零。只处理 `[1..m-1][1..n-1]` 的区域，不动第一行第一列。

```java
if (firstRowZero) { ... }
if (firstColZero) { ... }
```

最后处理第一行和第一列本身。必须放最后，否则会破坏标记。

## 复杂度分析

### 方法一：额外数组

时间复杂度：`O(m × n)`。两遍遍历。

空间复杂度：`O(m + n)`。两个标记数组。

### 方法二：原地标记

时间复杂度：`O(m × n)`。常数遍次数的遍历。

空间复杂度：`O(1)`。只用了两个布尔变量。

## 常见坑

### 1. 步骤 3 先处理了第一行/第一列

如果在步骤 3 就把第一行/第一列清零，标记信息就丢了，后面就无法正确判断哪些行列该清零。

必须最后处理第一行第一列。

### 2. 步骤 2 从 (0,0) 开始遍历

步骤 2 应该从 `(1,1)` 开始，因为第一行第一列是标记位，不应该在这一步被当作"数据"处理。

### 3. 分不清谁标记谁

`matrix[i][0] = 0` 表示第 `i` 行需要清零（第一列做行标记）。

`matrix[0][j] = 0` 表示第 `j` 列需要清零（第一行做列标记）。

## 面试时怎么讲

### 30 秒版

> 不能边扫边置零，否则新的零会引发连锁。先用第一行和第一列当标记位记录哪些行列要清零，再根据标记统一置零，最后单独处理第一行和第一列。时间 `O(m*n)`，空间 `O(1)`。

### 面试官可能追问

**为什么要先检查第一行第一列？**

因为它们被征用做标记位了。如果不先记录它们自身是否有 0，标记过程会覆盖原始信息，就分不清标记位上的 0 是"原本就是 0"还是"被标记的"。

**能不能用一个特殊值代替 0 来标记？**

题目没有限定元素范围，如果用特殊值（比如 `Integer.MIN_VALUE`），可能和数据冲突。用第一行第一列做标记是更通用的方案。

## 最后总结

矩阵置零的核心挑战是避免"连锁清零"：

> 先标记后操作，标记位复用矩阵自身的第一行第一列。

代码模板：

```text
记录第一行/第一列是否有 0
遍历剩余部分，把 0 的标记写到第一行/第一列
根据标记清零剩余部分
最后清零第一行/第一列
```

最后把这句话记住：

> 不能立刻动手——先把"谁该清零"全部记下来，最后一起执行。标记位就藏在矩阵自己身上。
