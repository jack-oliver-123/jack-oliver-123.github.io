## 树形结构

### 什么是树

树是一种**非线性**的数据结构，由 n（n≥0）个有限结点组成一个具有层次关系的集合。叫做"树"是因为它看起来像一棵倒挂的树——根朝上，叶朝下。

树的特点：

- 有一个特殊的结点称为**根结点**，根结点没有前驱
- 除根结点外，其余结点被分成 M（M>0）个互不相交的集合，每个集合又是一棵子树
- **树是递归定义的**——这一点决定了后续几乎所有操作都用递归实现

> 注意：子树之间不能有交集，否则就不是树形结构（而是图）。

### 树的核心概念

| 概念 | 含义 | 示例 |
|------|------|------|
| **结点的度** | 该结点子树的个数 | 有 3 个孩子 → 度为 3 |
| **树的度** | 所有结点度的最大值 | 最多的那个结点有 3 个孩子 → 树的度为 3 |
| **叶子结点** | 度为 0 的结点（没有孩子） | 树的末端节点 |
| **父结点** | 有子结点的结点 | A 有孩子 B、C → A 是父结点 |
| **子结点** | 某结点的下一层直接后继 | B、C 是 A 的子结点 |
| **根结点** | 没有父结点的结点（整棵树只有一个） | 最顶部的结点 |
| **结点的层次** | 根为第 1 层，往下递增 | 根的孩子在第 2 层 |
| **树的高度/深度** | 结点的最大层次 | 有 4 层 → 高度为 4 |
| **森林** | m（m≥0）棵互不相交的树的集合 | 删掉根后，子树构成森林 |

### 树的表示——孩子兄弟表示法

树的存储方式有多种（双亲表示法、孩子表示法等），最常用的是**孩子兄弟表示法**：

```java
class TreeNode {
    int val;
    TreeNode firstChild;    // 第一个孩子
    TreeNode nextSibling;   // 下一个兄弟
}
```

这种方式的巧妙之处：**任意一棵树都可以用二叉树来表示**（第一个孩子作为左子树，兄弟作为右子树）。

### 树的应用

- **文件系统**：目录和文件天然是树形结构
- **DOM 树**：HTML 文档的结构
- **组织架构**：公司的层级关系
- **数据库索引**：B 树、B+ 树

## 二叉树

### 定义

一棵二叉树是结点的一个有限集合，该集合：

- 或者为**空**
- 或者由一个**根结点**加上两棵分别称为**左子树**和**右子树**的二叉树组成

关键特征：

1. 每个结点**最多**有两个子树（度 ≤ 2）
2. 子树有**左右之分**，次序不能颠倒（二叉树是**有序树**）

任意二叉树都是以下五种基本形态的组合：

```
  空树    只有根    只有左子树   只有右子树   左右都有
  
  NULL      A          A           A          A
                      /             \        / \
                     B               B      B   C
```

### 两种特殊的二叉树

#### 满二叉树

每一层的结点数都达到最大值。如果层数为 K，则总结点数为 **2^K - 1**。

```
        A           第 1 层：1 个结点
       / \
      B   C         第 2 层：2 个结点
     / \ / \
    D  E F  G       第 3 层：4 个结点
```

总共 2³ - 1 = 7 个结点。

#### 完全二叉树

对于深度为 K、有 n 个结点的二叉树，当且仅当其每一个结点都与深度为 K 的满二叉树中编号从 1 到 n 的结点一一对应时，称为完全二叉树。

通俗理解：**从上到下、从左到右依次填满**，最后一层可以不满，但必须从左到右连续。

```
完全二叉树 ✅           非完全二叉树 ❌
      A                       A
     / \                     / \
    B   C                   B   C
   / \                       \
  D   E                       D   （B 没有左孩子就有右孩子）
```

> **满二叉树是特殊的完全二叉树**——满二叉树的最后一层恰好也是满的。

### 二叉树的性质

这些性质在面试中经常考到，需要牢记：

| 编号 | 性质 | 说明 |
|------|------|------|
| 1 | 第 i 层最多有 **2^(i-1)** 个结点 | i ≥ 1 |
| 2 | 深度为 K 的树最多有 **2^K - 1** 个结点 | K ≥ 0 |
| 3 | **叶子数 = 度为2的结点数 + 1** | n₀ = n₂ + 1（最常考！） |
| 4 | n 个结点的完全二叉树深度为 **⌈log₂(n+1)⌉** | 向上取整 |
| 5 | 完全二叉树中，编号为 i 的结点（从 0 开始）：<br/>左孩子：2i+1<br/>右孩子：2i+2<br/>父结点：(i-1)/2 | 这是堆排序和优先队列的基础 |

**性质 3 的证明**（面试爱问）：

设总结点数 n = n₀ + n₁ + n₂（按度分类）。从"边"的角度看，除根结点外每个结点头上都有一条边，所以总边数 = n - 1。另外，度为 1 的结点贡献 1 条边，度为 2 的贡献 2 条边，所以总边数 = n₁ + 2·n₂。联立得 n₀ = n₂ + 1。

### 二叉树的存储

两种方式：

**1. 顺序存储（数组）**

用数组按层序存储，利用性质 5 的下标关系找父子——这就是**堆**的存储方式。适合完全二叉树，非完全二叉树会浪费空间。

**2. 链式存储（节点引用）**

每个节点存储数据和左右孩子的引用——这是最通用的方式：

```java
class TreeNode {
    int val;
    TreeNode left;   // 左孩子
    TreeNode right;  // 右孩子

    TreeNode(int val) {
        this.val = val;
    }
}
```

## 二叉树的遍历

遍历是二叉树最核心的操作——几乎所有二叉树问题都是遍历的变体。

### 前序遍历（Pre-order）

访问顺序：**根 → 左子树 → 右子树**

```java
// LeetCode #144
public void preOrder(TreeNode root) {
    if (root == null) return;
    System.out.print(root.val + " "); // 访问根
    preOrder(root.left);               // 递归左子树
    preOrder(root.right);              // 递归右子树
}
```

### 中序遍历（In-order）

访问顺序：**左子树 → 根 → 右子树**

```java
// LeetCode #94
public void inOrder(TreeNode root) {
    if (root == null) return;
    inOrder(root.left);                // 递归左子树
    System.out.print(root.val + " "); // 访问根
    inOrder(root.right);               // 递归右子树
}
```

> 对于**二叉搜索树**（BST），中序遍历的结果是有序的——这是中序遍历的一个重要性质。

### 后序遍历（Post-order）

访问顺序：**左子树 → 右子树 → 根**

```java
// LeetCode #145
public void postOrder(TreeNode root) {
    if (root == null) return;
    postOrder(root.left);              // 递归左子树
    postOrder(root.right);             // 递归右子树
    System.out.print(root.val + " "); // 访问根
}
```

### 层序遍历（Level-order）

从上到下、从左到右逐层访问。使用**队列（BFS）**实现：

```java
// LeetCode #102
public List<List<Integer>> levelOrder(TreeNode root) {
    List<List<Integer>> result = new ArrayList<>();
    if (root == null) return result;

    Queue<TreeNode> queue = new LinkedList<>();
    queue.offer(root);

    while (!queue.isEmpty()) {
        int size = queue.size(); // 当前层的结点数
        List<Integer> level = new ArrayList<>();

        for (int i = 0; i < size; i++) {
            TreeNode node = queue.poll();
            level.add(node.val);
            if (node.left != null) queue.offer(node.left);
            if (node.right != null) queue.offer(node.right);
        }
        result.add(level);
    }
    return result;
}
```

### 遍历示例

对于以下二叉树：

```
        A
       / \
      B   C
     / \   \
    D   E   F
```

- **前序**：A B D E C F（根左右）
- **中序**：D B E A C F（左根右）
- **后序**：D E B F C A（左右根）
- **层序**：A B C D E F（逐层）

> **记忆技巧**："前中后"指的是**根**的访问时机——前序根在前，中序根在中，后序根在后。左右的相对顺序始终不变（先左后右）。

## 二叉树的基本操作

这些操作都是遍历的变体，核心思路是**递归**——把问题分解为"对当前节点做什么 + 递归处理左右子树"。

### 求结点个数

```java
public int size(TreeNode root) {
    if (root == null) return 0;
    return 1 + size(root.left) + size(root.right);
}
```

### 求叶子结点个数

```java
public int leafCount(TreeNode root) {
    if (root == null) return 0;
    if (root.left == null && root.right == null) return 1; // 叶子
    return leafCount(root.left) + leafCount(root.right);
}
```

### 求第 K 层结点个数

```java
public int levelCount(TreeNode root, int k) {
    if (root == null) return 0;
    if (k == 1) return 1; // 到达目标层
    return levelCount(root.left, k - 1) + levelCount(root.right, k - 1);
}
```

### 求树的高度

```java
public int height(TreeNode root) {
    if (root == null) return 0;
    return 1 + Math.max(height(root.left), height(root.right));
}
```

### 查找值为 val 的结点

```java
public TreeNode find(TreeNode root, int val) {
    if (root == null) return null;
    if (root.val == val) return root;
    TreeNode left = find(root.left, val);
    if (left != null) return left;
    return find(root.right, val);
}
```

## 经典面试题

### 判断两棵树是否相同

```java
// LeetCode #100
public boolean isSameTree(TreeNode p, TreeNode q) {
    if (p == null && q == null) return true;
    if (p == null || q == null) return false;
    if (p.val != q.val) return false;
    return isSameTree(p.left, q.left) && isSameTree(p.right, q.right);
}
```

### 判断是否为对称二叉树

```java
// LeetCode #101
public boolean isSymmetric(TreeNode root) {
    if (root == null) return true;
    return isMirror(root.left, root.right);
}

private boolean isMirror(TreeNode left, TreeNode right) {
    if (left == null && right == null) return true;
    if (left == null || right == null) return false;
    return left.val == right.val
        && isMirror(left.left, right.right)
        && isMirror(left.right, right.left);
}
```

### 翻转二叉树

```java
// LeetCode #226
public TreeNode invertTree(TreeNode root) {
    if (root == null) return null;
    TreeNode temp = root.left;
    root.left = invertTree(root.right);
    root.right = invertTree(temp);
    return root;
}
```

### 判断是否为平衡二叉树

平衡二叉树：每个结点的左右子树高度差不超过 1。

```java
// LeetCode #110
public boolean isBalanced(TreeNode root) {
    return getHeight(root) != -1;
}

// 返回高度，如果不平衡返回 -1
private int getHeight(TreeNode root) {
    if (root == null) return 0;
    int left = getHeight(root.left);
    if (left == -1) return -1; // 左子树不平衡，直接剪枝
    int right = getHeight(root.right);
    if (right == -1) return -1;
    if (Math.abs(left - right) > 1) return -1;
    return 1 + Math.max(left, right);
}
```

### 最近公共祖先（LCA）

```java
// LeetCode #236
public TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
    if (root == null || root == p || root == q) return root;
    TreeNode left = lowestCommonAncestor(root.left, p, q);
    TreeNode right = lowestCommonAncestor(root.right, p, q);
    if (left != null && right != null) return root; // p 和 q 分居两侧
    return left != null ? left : right;
}
```

### 从前序与中序遍历构造二叉树

```java
// LeetCode #105
public TreeNode buildTree(int[] preorder, int[] inorder) {
    return build(preorder, 0, preorder.length - 1,
                 inorder, 0, inorder.length - 1);
}

private TreeNode build(int[] pre, int preL, int preR,
                       int[] in, int inL, int inR) {
    if (preL > preR) return null;
    TreeNode root = new TreeNode(pre[preL]); // 前序第一个是根
    // 在中序中找到根的位置
    int mid = inL;
    while (in[mid] != pre[preL]) mid++;
    int leftSize = mid - inL; // 左子树大小
    root.left = build(pre, preL + 1, preL + leftSize, in, inL, mid - 1);
    root.right = build(pre, preL + leftSize + 1, preR, in, mid + 1, inR);
    return root;
}
```

## 二叉树递归解题的思维模式

学会了上面这些题，你会发现二叉树的递归有一个固定模式：

```
1. 确定终止条件（通常是 root == null）
2. 确定当前节点要做什么
3. 递归处理左右子树
4. 利用左右子树的返回值得出当前节点的答案
```

这个模式可以覆盖 80% 以上的二叉树面试题。关键在于想清楚：**这个问题能不能分解为"左子树的答案 + 右子树的答案 + 当前节点的信息"？** 如果能，就可以递归解决。

## 小结

| 概念 | 核心要点 |
|------|---------|
| 树 | 非线性、层次结构、递归定义 |
| 二叉树 | 度 ≤ 2、有序（左右之分） |
| 满二叉树 | 每层都满，n 层有 2^n - 1 个结点 |
| 完全二叉树 | 从上到下从左到右连续填充 |
| 前/中/后序遍历 | 根的访问时机不同，都是 DFS |
| 层序遍历 | BFS，用队列实现 |
| 递归解题模式 | 终止条件 + 当前操作 + 递归左右 + 合并结果 |

**关键认知**：

- 二叉树的本质是**递归结构**——左子树和右子树本身也是二叉树
- 几乎所有二叉树操作都可以用递归实现，递归的关键是**信任子问题的结果**
- 完全二叉树的编号性质（父子下标关系）是堆和优先队列的基础
- 中序遍历 + BST = 有序序列——这是 TreeMap/TreeSet 的核心原理
