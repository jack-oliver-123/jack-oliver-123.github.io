## 题目入口

题目链接：[转置矩阵](https://leetcode.cn/problems/transpose-matrix/)

## 一句话读懂题目

给你一个 `m × n` 的矩阵，返回它的转置矩阵。

转置就是把行变成列、列变成行：原矩阵的第 `i` 行第 `j` 列，变成新矩阵的第 `j` 行第 `i` 列。

比如：

```text
mat = [[1, 2, 3],
       [4, 5, 6],
       [7, 8, 9]]
```

转置后：

```text
[[1, 4, 7],
 [2, 5, 8],
 [3, 6, 9]]
```

非方阵也行：

```text
mat = [[1, 2, 3],
       [4, 5, 6]]
```

2×3 矩阵转置后变成 3×2：

```text
[[1, 4],
 [2, 5],
 [3, 6]]
```

## 先用生活例子理解

想象你有一张 Excel 表格，3 行 2 列。

转置就是把"行标题变成列标题，列标题变成行标题"——相当于把表格沿左上到右下的对角线翻折。

原来 (第2行, 第1列) 的数据，现在跑到了 (第1行, 第2列)。

## 核心思路

转置的规则只有一条：

```text
result[j][i] = mat[i][j]
```

新矩阵的大小是 `n × m`（行列互换）。

原矩阵 2×3，转置后 3×2。

| (0, 1) | `2` | (1, 0) |

| (1, 1) | `5` | (1, 1) |
| (1, 2) | `6` | (2, 1) |

```text
 [2, 5],

```

## Java 代码

先记住一句话：

> 行变列，列变行，坐标交换就是转置。

```java
class Solution {
    public int[][] transpose(int[][] matrix) {
        int m = matrix.length;
        int n = matrix[0].length;
        int[][] result = new int[n][m];

        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                result[j][i] = matrix[i][j];
            }
        }

        return result;
    }
}
```

## 代码逐行拆解

```java
int m = matrix.length;
int n = matrix[0].length;
```

原矩阵 `m` 行 `n` 列。

```java
int[][] result = new int[n][m];
```

转置后的矩阵是 `n` 行 `m` 列。这里行列互换了。

```java
for (int i = 0; i < m; i++) {
    for (int j = 0; j < n; j++) {
        result[j][i] = matrix[i][j];
    }
}
```

遍历原矩阵每个元素，把 `(i, j)` 位置的值放到新矩阵的 `(j, i)` 位置。

## 复杂度分析

时间复杂度：`O(m × n)`。

每个元素访问一次。

空间复杂度：`O(1)`。

除了结果矩阵外没有额外空间。

## 常见坑

### 1. 新矩阵的大小写反了

新矩阵是 `n × m`，不是 `m × n`。对方阵来说没区别，但非方阵写反会数组越界。

### 2. 混淆转置和旋转

转置是沿对角线翻折，旋转是顺时针/逆时针转 90°。

- 转置：`result[j][i] = mat[i][j]`
- 顺时针旋转 90°：转置 + 左右翻转

### 3. 试图原地转置非方阵

非方阵不能原地转置，因为行列数不同，矩阵形状会变。只有方阵可以原地转置（交换 `mat[i][j]` 和 `mat[j][i]`）。

## 面试时怎么讲

### 30 秒版

> 转置就是行列互换。创建一个 `n × m` 的新矩阵，把原矩阵 `(i, j)` 位置的值放到 `(j, i)` 位置。时间 `O(m*n)`，空间 `O(1)`。

### 面试官可能追问

**方阵能不能原地转置？**

可以。只遍历上三角（`j > i` 的部分），交换 `mat[i][j]` 和 `mat[j][i]`。

```java
for (int i = 0; i < n; i++) {
    for (int j = i + 1; j < n; j++) {
        int temp = mat[i][j];
        mat[i][j] = mat[j][i];
        mat[j][i] = temp;
    }
}
```

**转置和旋转的关系？**

顺时针旋转 90° = 转置 + 每行左右翻转。逆时针旋转 90° = 转置 + 每列上下翻转。

## 最后总结

转置矩阵是矩阵操作中最基础的一道：

> 记住一个公式：`result[j][i] = mat[i][j]`。

代码模板：

```text
创建 n×m 的新矩阵
遍历原矩阵 (i, j)
    新矩阵 [j][i] = 原矩阵 [i][j]
```

最后把这句话记住：

> 转置的本质是坐标交换。行号变列号，列号变行号，仅此而已。
